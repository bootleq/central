Firefox 系統需求

Windows

作業系統版本
  • Windows 98
  • Windows 98SE
  • Windows ME
  • Windows NT 4.0
  • Windows 2000
  • Windows XP (推薦使用此版本)
硬體最低需求
  • Pentium 233 MHz (建議 Pentium 500MHz 以上)
  • 64 MB RAM (建議 128 MB 以上)
  • 52 MB 硬碟空間

Linux

軟體需求

These are currently under review and should be used as a general guideline.
請留意：Linux 隨附的 Package 依發佈版本亦可能有不同需求。

  • Linux kernel - 2.2.14，最少需安裝以下函式庫：
    • glibc 2.2.4
    • gtk+ - 1.2.0 (最好為 1.2.5 以上版本)
    • XFree86-3.3.6
    • fontconfig (亦稱為 xft)
    • gtk+2.0
  • Firefox 已於 Red Hat Linux 8.0 以上版本測試過。
硬體最低需求
  • Intel Pentium II 或 AMD K6-III+ 233 MHz CPU (建議 500MHz 以上)
  • 64 MB RAM (建議 128 MB 以上)
  • 52 MB 硬碟空間

Mac

作業系統版本
  • Mac OS X 10.1.x
  • Mac OS X 10.2.x and later
Minimum Hardware
  • PowerPC 604e 266 MHz (建議 PowerPC G4 667MHz 以上)
  • G3, G4, 64 MB RAM (建議 256 MB 以上)
  • 72 MB 硬碟空間
