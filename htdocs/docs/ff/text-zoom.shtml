放大／縮小文字

您經常抱怨網頁的字體太小嗎？家中有長輩視力不太好嗎？Firefox 的多段式文字縮放功能正可滿足您的需求。遇到文字太小的網頁，只要按住 Ctrl 搭配「加號鍵」（+）便可放大文字；文字太大時可以按住 Ctrl 搭配「減號鍵」縮小文字。您甚至可以將文字放大至全螢幕寬，當然放得這麼大沒什麼意義就是了 :P

實際瞧瞧

想像您以投影機播放網頁畫面，希望讓坐在最後面的人也能看清楚網頁上的字。這時，只要按住 Ctrl 並按加號鍵數次，Firefox 可以幫您將文字放大，要多大有多大。

進階技巧

以滑鼠縮放文字

如果您的滑鼠擁有滾輪，則按住 Ctrl 後也可滾動滑鼠滾輪，以便縮放文字。

回復原狀

縮／放得太多，不知道怎麼才能快速回復原狀？只要按Ctrl+0 就可以囉！
