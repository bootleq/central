Mozilla Taiwan 相關軟體中文測試版

歡迎您協助測試此處的中文軟體，您的貢獻能讓我們更好。

請注意：這裡列出的版本尚未正式發行，僅供測試用，一般使用者請下載正式版本。使用測試版軟體可能會有各種相容性問題、套件及佈景主題亦可能失效。

注意事項

Mozilla Taiwan 為提昇軟體中文翻譯品質，提供下列軟體中文測試版。

  • 如軟體操作時發生錯誤或有建議，請至 Buzilla 回報。
  • 若發現任何中文翻譯錯誤或建議，請至 Mozilla Taiwan 討論區回報。

本區檔案來源為 Mozilla.org Nightly Builds (每日編譯供測試的版本，非正式發行用)：

  • Firefox L10n Nightly
  • Thunderbird L10n Nightly

Firefox 1.5 Beta 1 (測試版，0922 Build)

  • Windows: 自解安裝檔、ZIP 免安裝
  • Linux: ZIP 免安裝
  • Mac OS X: DMG，解開後拖至應用程式內即可

Thunderbird 1.5 Beta 1 (測試版，0921 Build)

  • Windows: 自解安裝檔、ZIP 免安裝
  • Linux: ZIP 免安裝
  • Mac OS X: DMG，解開後拖至應用程式內即可

關於 Firefox 1.5 Beta

Firefox 1.5 正式版預計於今年底推出，目前測試版提供擴充套件作者及網頁設計師測試相容問題，其他相關資訊請見 Mozilla Developer Center 上的文章。

關於 Thunderbird 1.5 Beta

Thunderbird 1.5 正式版推出時間未定，目前測試版提供擴充套件作者及社群成員測試相容及使用問題，其他相關資訊請見 Thunderbird 1.5 Beta 1 Release Note 上的文章。
