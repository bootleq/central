Internet Explorer 8 是 Windows Internet Explorer 系列的最新版本，如果您不曾用過 Firefox，那麼它提供了更多人性化、個人化、隱私保護的新功能，您的網路生活終於可以注入新體驗。

利用附加元件庫自訂瀏覽體驗

Firefox 附加元件網站蒐集上千種附加元件，供您自訂瀏覽體驗。 在為您的 Firefox 加裝免費的 Firefox Activities 之後，您更可以到微軟提供的 IE8 附加元件庫，安裝最新的 IE8 加速器，立刻實現世界大同本一家的夢想。

熱門功能

使用加速器更快地完成工作

只要按幾下滑鼠，就可以檢視行駛路線、翻譯文字、在網路上將您搜索的內容以電子郵件形式傳送給其他人，以及其他功能。

利用Firefox Activities 擴充套件的「加速器」功能，更快速地完成每日的瀏覽活動。

立即試用這些加速器

  • URMAP 你的地圖 - 安裝 URMAP 加速器，在網頁上圈選想查詢的地址，就可以快速取得該地址的地圖。
  • 尋找其他有用的「加速器」

順利瀏覽過時網頁

Firefox 現在可以正確顯示專為過時瀏覽器（例如IE6）所設計的網站。如果您看到網站發生顯示問題，例如文字、影像或文字方塊未對齊，只要安裝 2004 年公開的 IE Tab 擴充套件，再按下右下角的「IE」按鈕，即可使用 IE 核心順利瀏覽這些過時的 IE Only 網頁。

IE Tab 僅能在Windows作業系統下使用。

使用網頁快訊隨時掌控最重要的資訊

只要新增網頁快訊，您可以不必一再地回到相同網站，即可取得新聞、股票報價、線上拍賣、天氣預報，甚至比賽分數的更新資訊。 利用 WebChunks 的「網頁快訊」功能，在「書籤工具列」下方顯示您所關心的最新資訊。使用 WebChunks，您甚至不必等網站服務商提供網頁快訊，可以自行「切」出想看的內容！

本功能仍屬測試中，歡迎協助測試回報問題。

阻擋壞網站，保全您的網路體驗

古老的「SmartScreen 篩選器」可透過保護您免受詐騙和惡意網站的攻擊，從而避免洩漏您的資料、隱私權和身份，讓您更安全地瀏覽和收發電子郵件。

2006 年公開的 Firefox 2 早已內建此項防惡意網站功能，使用Google及PhishTank提供的惡意網站資訊保全您的網路體驗。

速度快

是的！只要不開啟如網路地圖、文件、撲浪等有趣活潑的 Web 2.0 網站，Internet Explorer 8 顯示網頁的速度跟其他瀏覽器都差不多快 —— 甚至在某些網站比 Firefox 快 0.02 秒！即使將目前網站必備的「JavaScript」執行速度也列入考量，根據 CNET 的測試 (縱軸為時間，越短越快)、在五大瀏覽器中 IE8 仍有第五名的優秀表現。多花 Firefox 3 的 30% 時間，您一樣可以享受豐富的網路體驗。
