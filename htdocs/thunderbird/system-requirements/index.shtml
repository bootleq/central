Thunderbird 系統需求

Windows

作業系統版本
  • Windows 98
  • Windows 98SE
  • Windows ME
  • Windows NT 4.0
  • Windows 2000
  • Windows XP (推薦使用此版本)
  • Windows Vista
硬體最低需求
  • Pentium 233 MHz (建議 Pentium 500MHz 以上)
  • 64 MB RAM (建議 128 MB 以上)
  • 52 MB 硬碟空間

Linux

軟體需求
請留意：Linux 隨附的 Package 依發佈版本亦可能有不同需求。
  • Linux kernel - 2.2.14，最少需安裝以下函式庫：
    • glibc 2.3.2
    • libstdc++5
    • XFree86-3.3.6
    • fontconfig (亦稱為 xft)
    • gtk+2.0
  • Thunderbird 已於 Linux Fedora Core 4 測試過。
硬體最低需求
  • Intel Pentium II 或 AMD K6-III+ 233 MHz CPU (建議 500MHz 以上)
  • 64 MB RAM (建議 128 MB 以上)
  • 52 MB 硬碟空間

Mac

作業系統版本
  • Mac OS X 10.2.x 以上
硬體最低需求
  • 麥金塔電腦，核心為 Intel x86 或 PowerPC G3-G5 處理器
  • 128 MB RAM (建議 256 MB 以上)
  • 200 MB 硬碟空間
