使用 Camino 的理由

“It's brisk, baby.” - Om Malik of Business 2.0

“Best. Browser. Ever.”- Justin Collins

“Camino offers top-notch performance, elegant tabbed browsing, and an easy way to view and hide histories and bookmarks.” - John Rizzo, c|Net

Gecko 佈局引擎
Camino 使用與 Firefox 相同的 Gecko 佈局引擎，支援最新的 W3C 標準，能給您最佳的瀏覽體驗。
Mac OS X 原生介面
Camino 以 Cocoa 使用者介面撰寫而成，並針對 Mac OS X 做了最佳化處理。能與 Mac OS X 緊密結合。
友善的操作介面
Camino 重新設計過的操作介面使瀏覽網路變的非常簡單，並內建所有新世代瀏覽器必備的分頁瀏覽與彈出型視窗阻擋等功能。相關資訊...
方便的 Google 搜尋
在工具列中便已內建 Google 搜尋功能，讓您彈指之間找到您想要瀏覽的網站。
隱私及安全
Camino 能讓您用簡單的方式管理或刪除您在網路上留下的 Cookies、歷史紀錄、快取及下載的檔案。
更有效率
Camino 提供了方便的鍵盤快速鍵供您使用。遇上需要登入的網頁，Camino 也能自動幫您填入所需的登入資訊。

系統需求

  • Mac OS X 10.1.5 或更新版本
  • Mac OS X 10.1.5 的使用者需要下載 MRJPlugin 以使 Java 能正常使用
  • 128 MB 記憶體
  • 30 MB 可用的磁碟空間
  • 0.9a1 與逐日建製版需要 Mac OS X 10.2 或更高版本

Camino 是技術預覽版

Camino 為「技術預覽」版。即使本軟體性能良好，在大多數情形下足以成為您的主要瀏覽器，但我們不對其性能與可靠性做任何保證。這是尚未完工的版本，不應在重要關鍵作業上使用。詳情請見 [授權協議]。
