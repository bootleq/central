Firefox 3: 不會不上手，會讓您愛不釋手

  • 更快速的網路服務

    Firefox 3的開啟網頁比以往更流暢、更快速，不用再發呆數小橫槓，讓您遨遊網海更有效率。

  • 阻擋廣告視窗更有效

    內建阻擋彈出視窗功能，煩人的彈出視窗將不會打擾您愉快的心情。

  • 更順眼的風格介面

    Firefox 3新一代的介面，如同變色龍般神奇，不論是任何的作業系統與風格主題，都能夠做出最適合的配搭。您更可以透過下載主題，打造個人風格的Firefox。

  • 分頁瀏覽

    使用這個節省時間的功能，您可以在同一視窗內觀看多個網頁。將連結開啟於背景分頁之中，當您準備好要閱讀其他頁面時，它們已經載入完畢，您不需花費時間等待。相關資訊...

  • 維護您的隱私及安全

    Firefox內建多種隱私保護工具，不論是密碼管理或是惡意網站的過濾都更加升級，讓您在網路的世界不用擔心多餘的威脅。

  • 更聰明的搜尋功能

    網址列中內建多種入口網站的搜尋功能，「搜尋欄」中的「搜尋建議」等貼心功能仍隨侍在列，幫助您更快的找到你所需要的資訊。

  • 即時書籤

    整合RSS，方便您訂閱最新頭條及網站更新訊息。在瀏覽網頁時，還可以按下網址列上的「星」型小圖示，快速地把網頁加入收藏、加到資料夾及加入tag 等。

  • 系統資源更節省

     Firefox 3一改過去開多個分頁或視窗會增加系統資源消耗，使執行速度變慢的困擾，『特性指引最佳化』能依據內部執行的方式不同而產生最佳化的Firefox 執行檔；『記憶體再生器』能隨時釋放不用的記憶體。您將不會感受到Firefox對您的電腦有任何拖累。

  • 下載檔案更便利

    您可以任意調整檔案下載的位置，下載管理員具備有查找功能，幫助您掌握下載的狀況，而且不減您的使用效率。

  • 完全符合您的使用習慣

    Firefox 3的設定有最大的自由度，您可以任意的調整書籤、工具列、狀態列、添加按鈕等等。使Firefox符合您習慣的完整功能：想要全螢幕還可以自由縮放文字，符合您閱讀上的需求。
